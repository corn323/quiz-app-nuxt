## 專案初始化與安裝
```bash
# 使用 pnpm 安裝相依套件
pnpm install

# 執行 Prisma 資料庫遷移與產生 Prisma Client
pnpm prisma migrate dev
pnpm prisma generate

# 專案啟動與開發者頁面
pnpm run dev

# 專案建置
pnpm run build
```
